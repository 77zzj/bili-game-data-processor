# B站游戏视频数据自动化提取脚本 (Bili-Game-Data-Processor)

这是一个基于 Python 的实用脚本。主要目的是为了解决在构建游戏内容分析平台时，抓取到的视频文本数据太杂乱、难以直接入库的问题。

## 实现逻辑
1. **数据抓取模块**：模拟请求获取目标游戏（如《原神》、《绝区零》）的视频基础元数据（BVID、标题、简介）。
2. **大模型清洗模块**：通过构造 Prompt，调用开源大模型 API 对“标题+简介”进行语义分析，提取出结构化的标签（如：玩法分类、核心术语关键词）。
3. **数据落盘**：将原始非结构化文本与模型打标结果合并，输出为干净的 JSON 格式，方便后续直接导入 MySQL 或向量数据库（如 FAISS）。

## 依赖与运行环境
- Python 3.8+
- `requests`

## 运行方法
1. 在环境变量中配置大模型 API 密钥：
   `export MY_LLM_API_KEY="你的密钥"`
2. 运行脚本：
   `python bili_processor.py`
